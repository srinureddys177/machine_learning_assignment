# Evaluating k-Means Clustering Across Different Dataset Structures

This project demonstrates how k-Means clustering behaves on datasets with different shapes and structures.

## Datasets
- Blobs
- Moons
- Iris

## Main goals
- Explore the effect of the number of clusters (k)
- Use elbow plots and silhouette scores
- Compare clustering behaviour across datasets
- Show strengths and limitations of k-Means

## Files
- `notebook.ipynb` - main analysis notebook
- `dataset_comparison_results.csv` - summary of selected results
- `all_kmeans_results.csv` - all k values tested
- `requirements.txt` - dependencies
- `tutorial.pdf` - final tutorial submission

## How to run
1. Install dependencies:
   `pip install -r requirements.txt`

2. Start Jupyter:
   `jupyter notebook`

3. Open the notebook and run all cells.
